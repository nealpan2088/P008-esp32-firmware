# P008 — ESP32 + A7670C 短信报警网关

通过 MQTT 接收后端报警消息，驱动 A7670C 4G 模块发送短信到客户手机。

## 架构

```
传感器报警 → 后端检测 → MQTT (p008/sms/alert)
                              ↓
                     家里的 ESP32 (MQTT 订阅)
                              ↓
                      A7670C → 发短信到客户手机
                              ↓
                     MQTT 回执 (p008/sms/status)
```

## 硬件清单

| 硬件 | 数量 | 说明 |
|------|------|------|
| ESP32 开发板 | 1 | 主控，连 WiFi + MQTT |
| A7670C 4G 模块 | 1 | 发短信，需要 2A 独立供电 |
| SIM 卡 | 1 | 支持 4G 的普通手机卡 |
| 4G 天线 | 1 | 模块自带或另配 |

## 接线

> ⚠️ **A7670C 峰值电流可达 2A（4G 发射时），必须用独立 5V/2A 以上电源供电，严禁从 ESP32 的 5V 引脚取电。**

### 电源接线

```
              ┌──────────────────┐
              │   5V/2A+ 电源适配器   │
              └────────┬─────────┘
                       │
                  ┌────┴────┐
                  │  5V (+) │────── A7670C VCC
                  │  5V (+) │────── ESP32 VIN (如果 ESP32 也用这个电源)
                  │  GND (-)│────── A7670C GND
                  │  GND (-)│────── ESP32 GND
                  └─────────┘
```

**方案A（推荐）— 共用一个 5V/3A 电源：**
- 5V/3A 电源正极 → 同时接 **A7670C VCC** 和 **ESP32 VIN**
- 5V/3A 电源负极 → 同时接 **A7670C GND** 和 **ESP32 GND**
- ✅ 共地、共电，接线简单

**方案B — 分开供电：**
- A7670C 用独立 5V/2A 电源，ESP32 用 USB 线连电脑或自己的电源
- ⚠️ 两个电源的 GND 必须连通（万用表确认共地）

### 信号接线

| ESP32 引脚 | 连接 | A7670C 引脚 |
|-----------|------|-------------|
| GPIO16 (RX2) | ← | TX（模块发，ESP32 收）|
| GPIO17 (TX2) | → | RX（ESP32 发，模块收）|
| GND | ↔ | GND（必须共地）|

### A7670C 模块其他必要引脚

| A7670C 引脚 | 接法 |
|------------|------|
| **VCC** | 5V/2A+ 独立电源正极 |
| **GND** | 电源负极，同时接 ESP32 GND |
| **PWR_KEY** | 经 10KΩ 电阻接模块 VCC（或直接短接，模块上电自动开机）|
| **RST** | 悬空（模块内部已上拉）|
| **VDD_EXT** | 悬空（内部 1.8V 输出）|

### 接线步骤

1. **先断电**，确保电源和 ESP32 USB 都拔掉
2. 接 **A7670C 电源线**：VCC→电源正极，GND→电源负极
3. 接 **共地线**：A7670C GND ↔ ESP32 GND
4. 接 **信号线**：ESP32 GPIO16 → A7670C TX，ESP32 GPIO17 → A7670C RX
5. 接好 **PWR_KEY**：通过 10KΩ 电阻连到 VCC（或直接短接）
6. 装上 **SIM 卡** 和 **天线**
7. **先给 A7670C 通电**，等 2 秒再插 ESP32 USB
8. 检查 A7670C 指示灯：
   - 慢闪（1秒/次）= 模块已开机，正在搜网
   - 快闪（~1次/200ms）= 模块在 4G 网络注册中
   - 快闪后常亮或 3 秒慢闪 = 网络注册成功 ✅

### ⚠️ 常见错误

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| ESP32 5V 直接给 A7670C 供电 | 电压跌落 → 模块重启/发不出短信 | 独立 5V/2A+ 电源 |
| 没共地 | 串口通信乱码/AT无响应 | ESP32 GND ↔ A7670C GND |
| 信号线 RX/TX 接反 | 通信失败 | GPIO16→A7670C TX, GPIO17→A7670C RX |
| 没接 PWR_KEY | 模块不开机 | 10KΩ 接 VCC 或短接 |
| 没装天线 | 信号弱/注册不上网络 | 必须装 4G 天线 |

## MQTT Topic

| Topic | 方向 | 说明 |
|-------|------|------|
| `p008/sms/alert` | 后端 → ESP32 | 报警短信内容 |
| `p008/sms/status` | ESP32 → 后端 | 发送状态回执 |

## 报警消息格式

```json
{
  "to": "13800138000",
  "message": "【睿云智感】冷库A温度超标！当前: 12.5°C",
  "id": "alert-xxx"
}
```

## 编译上传与配网

```bash
cd esp32-sms-gateway
pio run -t upload
```

**首次使用**：烧录后 ESP32 会开一个 WiFi 热点 `P008-SMS-Gateway`（密码 `config123`）：
1. 手机/电脑连这个热点
2. 浏览器会自动弹出配置页面（不会弹出就访问 `192.168.4.1`）
3. 选择你家 WiFi，输入密码
4. 保存后 ESP32 自动重启连接 WiFi

**后续更换 WiFi**：长按某个 GPIO 引脚可重置配网（暂未实现，可用 `ESP.restart()` 或清除 flash 方式重置）。
