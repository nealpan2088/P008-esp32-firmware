# P008 环境监测 - A7670C 4G 通知模块

## 用途
作为微信推送（WxPusher）的**保底方案**，当客户微信消息触达不到时，通过 **短信/电话** 直接通知客户。

## 工作流程

```
传感器异常报警
    ↓
P008 后端（服务器）
    ├── 发送 WxPusher 微信推送（主力方案）
    │
    └── 如果客户微信不理想/重要报警
        ↓
    A7670C 模块（4G）
        ├── 发短信 → 客户手机收到
        └── 打电话（TTS 语音）→ 客户接听听到语音提醒
```

## 硬件连接

| A7670C 模块 | Arduino/ESP8266 |
|:-----------:|:---------------:|
| TX | RX (GPIO4/D2) |
| RX | TX (GPIO5/D1) |
| VCC | 5V（建议独立电源）|
| GND | GND |
| 喇叭（TTS用） | SPK+ / SPK- |

## 供电注意
- A7670 瞬间电流可达 **2A**，建议独立 5V/2A 电源供电
- 不要从 Arduino/ESP8266 的 5V 引脚取电（不够用）

## API 对接（后端需要新增的接口）
- `GET /api/v1/notify/pending?device=xxx` - 获取待发送的短信队列
- `POST /api/v1/notify/callback` - 模块报告发送结果
